# Morning Brief

Use the bundled script to produce a daily morning brief (weather consensus + RSS-AI daily report) and post it as a Telegram **voice** message to a specific **forum topic**.

## Workflow

1. Prepare a config JSON (see `--init-config` output format in the script).
2. Run the script once manually to validate extraction + TTS + Telegram delivery.
3. Schedule it (cron) for your preferred time with `CRON_TZ=Asia/Shanghai`.

## Script

- Main entry: `scripts/morning_brief.py`

Examples:

```bash
python3 scripts/morning_brief.py --config examples/config.example.json --dry-run
python3 scripts/morning_brief.py --config /path/to/your/config.json
```

Notes:

- Weather summary comes from a multi-model consensus generated by Open-Meteo model outputs.
- Content policy: keep facts stable, avoid adding invented details, and make the final script suitable for voice playback.
